GFM是Github扩展的基于Markdown的一种纯文本书写格式
## 表格

	| Name | Academy | score | 
	| - | :-: | -: | 
	| Harry Potter | Gryffindor| 90 | 
	| Hermione Granger | Gryffindor | 100 | 
	| Draco Malfoy | Slytherin | 90 |

| Name | Academy | score | 
| - | :-: | -: | 
| Harry Potter | Gryffindor| 90 | 
| Hermione Granger | Gryffindor | 100 | 
| Draco Malfoy | Slytherin | 90 |
两边的`|`可以省略
`--`数量无所谓
`|:`右对齐
`:|`左对齐
`:|:`居中对齐
